2 Pick your fonts

Every project has three font roles — one for headlines, one for body text, and one for code. They can be the same font or three different fonts. Below the role pickers, choose where each font comes from.

The Figma reality: Whatever font you pick has to also be installed on your computer for it to show correctly in Figma. The browser can download fonts on the fly; Figma can't. We'll always tell you what to install — no guessing.

Quick check before you use a paid font

Fonts you buy or license have rules about where they can be used (your live website, your design files, your team). DTF doesn't check this for you — you're the one who picks, so you're the one who owns the call.

To make Inter look the same in Figma:
  1. Download Inter from fonts.google.com/specimen/Inter
  2. Open the zip, double-click the .ttf files, hit Install
  3. Restart Figma — it'll pick up the new font automatically
  4. Come back here and tick "Installed in Figma" so this reminder goes away

4 Set weight, line height & letter spacing

Three more ladders that finish off your type system: how thick text gets, how much breathing room between lines, and how tightly packed the letters are.

One small thing about letter spacing: In code we measure it in a unit called "em" — but Figma needs it in percent. DTF takes care of this for you in the background. You'll see both numbers below so it's never a surprise.

9 Decisions locked in (your input from this round)

What we've agreed so far. Anything not listed here is still on the table.

  1. Three font roles — headline, body, code. If a brand wordmark needs a fourth font, override it on just that one text style instead of adding a global slot.
  2. Fonts come from three places — built-in, Google Fonts, or your own. Each one shows a clear status chip so you always know whether it'll match in Figma. "Your own" needs a licence confirmation tick. Google Fonts shows a one-click install guide.
  3. Auto-generate size ladders with three options — type each size yourself, multiply by a fixed ratio, or resize with screen width. Pixel-based naming (size 14, size 16) matches how spacing already works.
  4. Save combinations as text styles (headline large, body base, caption…). These become Figma text styles automatically — same name, same meaning, both sides.
  5. Letter spacing stays in em in code, auto-converted to % on its way to Figma. Fixes a bug where Figma currently ignores letter spacing entirely.
  6. Per-text-style font override is allowed — so the brand wordmark can use a serif even though the rest of the system is sans.
  7. Live preview pane always visible — real paragraphs, real buttons, real alerts re-render as you tweak.
  8. Health checks run continuously — green / yellow / red callouts with plain-language suggestions, not technical jargon.
