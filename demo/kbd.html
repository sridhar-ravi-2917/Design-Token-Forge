Accessibility

A11y

KBD is a semantic HTML element. Screen readers announce it as keyboard input.

Semantic HTML Uses native <kbd> element. Screen readers announce content as keyboard input without extra ARIA.
Forced Colors In Windows High Contrast mode, border is enforced with ButtonText. Disabled uses GrayText.
Non-interactive KBD is display-only. No focus management, no tap target enforcement needed. --kbd-min-tap-target: 0px.
Contrast Text uses surface-base-ct-default (strong contrast). Ghost uses ct-subtle (still AA-passing).
